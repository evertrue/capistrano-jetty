# Capistrano::Jetty

This gem provides support in Capistrano 3.x for copying a deployed artifact into Jettyâ€™s webapps directory, then restart Jetty.

Plays nicely with [evertrue/capistrano-scm-jenkins](https://github.com/evertrue/capistrano-scm-jenkins).

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'capistrano-jetty'
```

And then execute:

```bash
$ bundle
```
